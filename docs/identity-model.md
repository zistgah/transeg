# Identity model — pointer
<!-- content CC-BY-SA-4.0 -->
The identity model (four typed layers, artifact envelope with provenance/consent/ownership/
signature/version/revocation, delegation and attestation, audit format) is normative in
zistgah/transeg-idgov. This core repo implements against those schemas and must track them.
Identity governance is not ecosystem governance: zistgah/governance governs contributions;
transeg-idgov governs identity. Separate concerns.
